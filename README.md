# 🪨 Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game built using **HTML, CSS, and Vanilla JavaScript**. Play against the computer, track scores in real time, and reset the game anytime. The application features a responsive interface with smooth interactions and random computer-generated moves.

---

## 🚀 Features

- 🎮 Player vs Computer gameplay
- 🎲 Random computer move generation
- 🪨 Rock, 📄 Paper, ✂️ Scissors selection
- 🏆 Automatic winner determination
- 📊 Live score tracking
- 🔄 Reset game functionality
- 📱 Responsive design for desktop and mobile
- 🎨 Clean and modern user interface

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## ▶️ Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/Rock-Paper-Scissors.git
```

2. Open the project folder.

3. Run `index.html` in your browser.

---

## 🎯 How to Play

1. Click **Rock**, **Paper**, or **Scissors**.
2. The computer randomly selects its move.
3. The winner is determined based on the classic game rules:
   - Rock beats Scissors
   - Scissors beats Paper
   - Paper beats Rock
4. Scores update automatically after every round.
5. Click **Reset Game** to start over.
